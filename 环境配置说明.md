# 运营助手浏览器 - 环境配置说明

## 环境区分

项目自动根据运行环境切换首页地址：

| 环境 | 判断条件 | 首页地址 |
|------|----------|----------|
| 开发环境 | `npm start` 运行 | `http://localhost:5173/` |
| 生产环境 | 打包后运行 | `https://dev.china9.cn/aigc_browser/` |

**原理**：通过 `app.isPackaged` 判断是否为打包后的应用。

## 常用命令

```bash
# 安装依赖
npm install

# 开发环境运行
npm start

# 打包生产版本
npm run build
```

## 常见问题

### 1. 第三方页面（抖音、小红书等）无法加载

**症状**：新窗口弹出但页面空白，只有注入脚本的横幅显示。

**原因**：通常是缓存冲突导致。

**解决方法**：
1. 关闭所有浏览器窗口
2. 删除缓存目录：
   ```
   C:\Users\Administrator\AppData\Roaming\custom-browser\Partitions\browserview\Cache
   C:\Users\Administrator\AppData\Roaming\custom-browser\GPUCache
   ```
3. 重新运行 `npm start`

### 2. Electron 安装失败

**症状**：`Electron failed to install correctly`

**解决方法**：
```bash
# 删除 node_modules 重新安装
rm -rf node_modules
npm install

# 如果下载失败，使用国内镜像
ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/ node node_modules/electron/install.js
```

### 3. 开发环境需要先启动 Vue 项目

运行 `npm start` 前，需要先在父级项目启动 Vue 开发服务器：

```bash
cd E:\项目\资海云\视频剪辑\ai-media-edit
npm run dev
```

确保 `http://localhost:5173/` 可以访问后，再启动本项目。

## 目录结构

```
D:\浏览器\运营助手\
├── main.js                 # 主进程（环境判断在此）
├── preload.js              # 控制面板预加载脚本
├── content-preload.js      # 内容页面预加载脚本
├── index.html              # 控制面板界面
├── script-manager.js       # 脚本管理器
└── injected-scripts/       # 注入脚本目录
    ├── scripts-config.json # 脚本配置
    ├── common.js           # 公共函数
    ├── douyin-creator.js   # 抖音授权脚本
    ├── douyin-publish.js   # 抖音发布脚本
    └── ...
```

## 缓存目录

应用数据存储在：
```
C:\Users\Administrator\AppData\Roaming\custom-browser\
├── Partitions\browserview\  # Session 数据、Cookie、缓存
└── GPUCache\                # GPU 缓存
```
