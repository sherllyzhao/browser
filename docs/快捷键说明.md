# 后门快捷键说明

本文档记录运营助手浏览器的后门快捷键功能。

## 快捷键列表

### 1. 打开开发者工具
**快捷键**: `Ctrl + Shift + F12`

**功能**:
- 打开当前聚焦窗口的 DevTools（开发者工具）
- 同时也会打开 BrowserView 的 DevTools
- 用于调试页面和查看控制台日志

**使用场景**:
- 需要查看页面元素
- 需要查看 console 日志
- 需要调试 JavaScript 代码
- 需要查看网络请求

---

### 2. 清除 Cookies（清除登录状态）
**快捷键**: `Ctrl + Alt + C`

**功能**:
清除指定网站或所有网站的登录状态（Cookies、LocalStorage、SessionStorage）

**操作步骤**:
1. 按 `Ctrl + Alt + C` 打开清除对话框
2. 选择清除范围：
   - **取消**: 取消操作
   - **清除所有登录状态**: 删除所有网站的登录信息
   - **清除指定域名**: 只删除特定网站的登录信息
3. 如果选择"清除指定域名"：
   - 在弹出的输入框中输入域名
   - 例如: `douyin.com` 或 `creator.douyin.com`
   - 只输入主域名(如 `douyin.com`)将清除所有相关子域名
4. 点击"清除"按钮或按回车确认
5. 等待成功提示框，显示删除了多少个 cookies
6. 刷新页面即可看到已退出登录

**使用场景**:
- 需要退出某个网站的登录状态
- 需要切换账号登录
- 测试登录流程
- 清除缓存数据

**示例域名**:
- `douyin.com` - 清除抖音所有相关网站的登录状态
- `xiaohongshu.com` - 清除小红书登录状态
- `creator.douyin.com` - 只清除抖音创作者平台登录状态
- `baijiahao.baidu.com` - 清除百家号登录状态
- `channels.weixin.qq.com` - 清除视频号登录状态

---

## 注意事项

1. **后门功能仅供开发和调试使用**，不要告诉最终用户
2. 清除所有登录状态后，所有网站都需要重新登录
3. 清除指定域名时，会匹配包含该域名的所有 cookies（包括子域名）
4. 快捷键在应用启动时会在控制台显示注册状态

## 查看日志

所有快捷键操作都会在控制台输出详细日志，格式如下：

```
[DevTools] 后门快捷键触发
[Clear Cookies] ========== 后门快捷键触发 ==========
[Clear Cookies] 显示选择对话框...
[Clear Cookies] 用户选择: 2
[Clear Cookies] 收到域名: douyin.com
[Clear Cookies] 开始清除域名: douyin.com
[Clear Cookies] 当前共有 47 个 cookies
[Clear Cookies] ✓ 删除: sessionid @ .douyin.com
[Clear Cookies] ========== 清除完成 ==========
[Clear Cookies] ✅ 共删除 15 个 cookies
```

查看日志的方法：
1. 在运营助手窗口按 `Ctrl + Shift + F12` 打开 DevTools
2. 切换到 Console（控制台）标签
3. 可以看到所有操作的详细日志
